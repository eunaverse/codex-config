# Weekly Template

## KPI Definitions

- Applications submitted: Count of roles applied this week.
- Resume-to-screen conversion: Screens / applications.
- OA pass rate: OA passed / OA received.
- System design score: Average of rubric scores (1-5).
- Behavioral score: Clarity + ownership + impact (1-5).

## Weekly Cadence

- Monday: Plan and JD selection.
- Tue-Thu: Core execution (coding + applications).
- Friday: System design mock and behavioral drill.
- Saturday: Portfolio/resume cleanup.
- Sunday: Review metrics and reset priorities.

## Review Prompts

- Which KPI regressed this week and why?
- Which lane produced highest interview ROI?
- What should be dropped next week to focus better?
