# Pattern Taxonomy

## Core Patterns

- Array and two pointers
- Sliding window
- Hashing and prefix sums
- Stack and monotonic stack
- Binary search and answer space search
- Tree DFS/BFS
- Graph traversal and topological sort
- Heap and greedy
- Backtracking
- Dynamic programming
- Intervals and sweep line

## Tracking Fields

- Pattern
- Problem ID
- Time to solve
- Failed reason
- Review due date
